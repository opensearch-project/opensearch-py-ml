Series.shape

Series.shape

Return a tuple representing the dimensionality of the Series.

Returns

shape: tuple

  1. number of rows

  2. number of columns

Notes

  • number of rows len(series) queries OpenSearch

  • number of columns == 1

Examples

>>> df = oml.Series(OPENSEARCH_TEST_CLIENT, 'ecommerce', name='total_quantity')
>>> df.shape
(4675, 1)
