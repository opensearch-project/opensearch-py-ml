groupby.DataFrameGroupBy.quantile

DataFrameGroupBy.quantile(q: int | float | List[int] | List[float] = 0.5) → pd.DataFrame[source]

Used to groupby and calculate quantile for a given DataFrame.

Parameters

q:

float or array like, default 0.5 Value between 0 <= q <= 1, the quantile(s) to compute.

Returns

pandas.DataFrame

quantile value for each grouped column

See Also

:pandas_api_docs:`pandas.core.groupby.GroupBy.quantile`

Examples

>>> from tests import OPENSEARCH_TEST_CLIENT

>>> oml_flights = oml_df.filter(["AvgTicketPrice", "FlightDelayMin", "dayOfWeek", "timestamp"])
