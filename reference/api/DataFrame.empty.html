DataFrame.empty

DataFrame.empty

Determines if the DataFrame is empty.

Returns

bool

If DataFrame is empty, return True, if not return False.

See Also

:pandas_api_docs:`pandas.DataFrame.empty`

Examples

>>> df = oml.DataFrame(OPENSEARCH_TEST_CLIENT, 'flights')
>>> df.empty
False
