DataFrame.to_html

DataFrame.to_html(buf=None, columns=None, col_space=None, header=True, index=True, na_rep='NaN', formatters=None, float_format=None, sparsify=None, index_names=True, justify=None, max_rows=None, max_cols=None, show_dimensions=False, decimal='.', bold_rows=True, classes=None, escape=True, notebook=False, border=None, table_id=None, render_links=False) → Any[source]

Render a OpenSearch data as an HTML table.

Follows pandas implementation except when max_rows=None. In this scenario, we set max_rows=60 to avoid accidentally dumping an entire index. This can be overridden by explicitly setting max_rows.
